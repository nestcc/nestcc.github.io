质数筛缺陷

线性筛思想

线性筛的总体思想是用一个整数M去标记合数N，其中N和M具有如下性质

  1. N中最小质数为P
  2. N可以表示为P*M
  3. P一定小于等于M中最小的因子
  4. 利用M（所有不大于M中最小的质数）去标记P*M
#define N 30

int nums[N + 5];

int init() {
	for (int i = 2; i < N; i += 1) {
		if (!nums[i]) { nums[++nums[0]] = i; }

		for (int j = 1; j <= nums[0]; j += 1) {
			if (nums[j] * i >= N) { break; }
			nums[nums[j] * i] = nums[j];
			if (i % nums[j] == 0) { break; }
		}
	}
	return 1;
}

需要判断质数的情况

在nums数组中，前nums[0]项已经被用于存放质数，因此我们在需要判断一个数字是否是质数的时候无法通过nums数组判断，因此我们需要另外一个prime数组。

int nums[N];
int prime[N] = { 1, 1, 0 };	// 务必将前三项重置为1，1，0

void init() {
    for (int i = 2; i < N; i += 1) {
        if (!nums[i]) { nums[++nums[0]] = i; }
        for (int j = 1; j <= nums[0] ; j += 1) {
            if (nums[j] * i >= N) { break; }
            nums[nums[j] * i] = i;
            prime[nums[j] * i] = i;
            if (i % nums[j] == 0) { break; }
        }
    }
    return ;
}
